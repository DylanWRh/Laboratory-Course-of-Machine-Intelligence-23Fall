4.客户端高级设置技巧

作者：王胜松
参考资料：

1 adjust computer to best run sl from Second Life For Dummies.pdf

2 第二人生客户端基本设置.doc


前言


相信通过前面几个章节的学习，您对里hippo opensim 的世界已经有了一次初步的体验。无论是惊奇赞叹还是摇头失望，我想都一定有你自己的理由。

或许你经常感觉到软件运行起来不很流畅，抑或感觉世界里的画面不够精致；或许你对软件的某些操作方式感到惊异，甚至不习惯；或许你从未注意到世界场景中的很多动人音乐；或许你对世界中交易，聊天等诸多系统有自己的想法与需求。那么恭喜你，本章正是为你的这些要求而准备的。

每个软件都尽量使自己能让有不同硬件环境的用户获得同样优秀的体验，尽管这实现起来很困难，但这正是我们一直所致力于的。我们已经为了设定了建议的默认值，这能够适合大部分主流的硬件环境，操作系统以及用户操作方式。但是如果你有更多自己的需求，或者在使用软件的过程中感觉到一些不愉快，那么请你阅读以下的内容。相信你经过合理的设置之后，一定能获得满意的体验。如果还不能使您满意，那么请联系我们。

我们的网站是 http://www.google.com/url?q=http%3A%2F%2Fwww.secondcampus.net.cn%2F&sa=D&sntz=1&usg=AFrqEzcaBSI_-YuLSSshnDAMvda1MLsYAg


第一节 您首先应该知道的事实

你应该拥有一台具有互联网连接能力的电脑。当然只要能够运行该软件即可，我们并不在意你是在普通pc上还是mac机，或者工作站，服务器上运行hippo opensim。

具有互联网连接能力包括了通过拨号，ISDN，ADSL，局域网，光纤或者无线等所有方式。这可能和你当地的运营商和你选择的服务有关，但只要你能正确连接到我们的服务器即可，唯一的差别在于速度的不同，而这可能正是让你头疼的问题。我们再后面会讲到这方面的设置。


官方配置要求

操作系统：Windows XP，Mac OS X v. 10.3.9 (Panther) or better

处理器：至少800MHz

内存：至少256MB

显卡：nVidia GeForce2，GeForce4 MX；and ATI Radeon 8500, 9250, or better


说明：对于Windows系列的操作系统，鉴于微软后续系统对旧版本的良好兼容，如果你拥有Windows XP SP1，2，3；Vista or sp1，甚至Windows 7的预览版本。或许你都可能正常运行我们的软件，据用户的反馈也的确有成功的例子。而如果是Windows 2000或者服务器版本的Windows，只要为Windows NT 4系列，你可以尝试运行并向我们反馈。至于Windows 9x系列，包括Windows Me，我们无法给出任何保证。


第二节 系统的设置

系统的设置请首先确保你已打开hippo opensim。然后点击Edit-Preferences，（或者直接Ctrl+P）呼出“个人偏好”界面。当然我们此处先谈能确保你流畅运行软件的基本设置。
让我们来看看哪些方面你可以大显身手。

第一个要谈到的是Network 选项卡。



Maximum Bandwidth:最大带宽值

这个滑动条允许你限制hippo opensim 能使用的最大带宽。我们建议在你的带宽和系统允许的情况下将该项调大，以便于你更快速流畅的享受hippo opensim的世界。

如果将该项设置过大，导致其占据了很大的带宽、cpu和内存，导致你的系统运行明显变慢而影响到你同时使用其它软件。你可以尝试将该项调小。

当然，如果你网络带宽本来就很小，或者距离服务器很远，将该项调大并不会明显的效果。

简单地说，一切取决于你的实际使用感受，一切使用感受取决于你的设置和你的系统和网络实际能力。

good luck！

Disk Cache Size:硬盘缓存大小

类似于浏览器，opensim 同样需要在磁盘上建立缓存来储存数据。缓存的作用相信不用我来讲。这里你最大可以设置为1000MB。如果你不小心将它拖到了最左边。￥%&￥##%￥

Clear Cache:清空缓存

通常情况下软件会自动清空并更新缓存。这里允许你手动清空。

在你未登录时，你可以点击这个图标





第三节 操作方式的设置

General tab 常规页

在Edit菜单下选择Preferences...或者直接用Ctrl+P，调出设置对话框

 

Start Location:登陆地点

你每次登陆Secondlife时所在的位置。有三个选项。

  • My Last Location - 上次退出时的位置。
  • My Home - 如果你拥有土地，将直接登陆到你的家。（选择World菜单下的Set Home to here来设置家的位置）
（Type region name）- 可以在这里输入区域名称。

Show Start Location on Login Screen

在登陆画面左下角显示登陆地点选择框。如图所示。


Show Names:显示名字

  • Never - 不显示他人的名字。
  • Show Temporarily - 短暂的显示他人名字。
  • Always - 一直显示他人名字。

Hide My Name On My Screen:在自己的画面上隐藏自己的名字

这并不影响其他人看到你的名字。

Hide My Group Title:隐藏组织名

其他人也将无法看到。

Small Avatar Names:小字体显示名字

Color For My Effects:效果色

当你选择一个物体时，手和物体间会出现由很多小球组成的光柱，这里的旋转就是用来改变该光柱的颜色。如下图中的蓝色小球。

Away Timeout:打盹时间

如果在一定时间不活动鼠标或键盘，系统将设置你为“Away”（打盹）状态。最大值为600秒。

Show Online Friend Notifications

好友上线或下线时，弹出一个提示框。

Rotate Mini-Map:旋转小地图

选择此项，小地图的上方就是你当前面对的方位。不选择的话，小地图为上北下南。

Notify when Linden dollars (L$) spent or received

当你钱数增加或减少时弹出一个提示框。

Auto-detect graphics hardware on next startup:每次登陆都自动检测硬件

Use default system color picker:使用系统颜色框

在选择颜色时，使用系统颜色框，如图。

不选择的话，使用Secondlife的颜色框，如图。

Crash reports:当机报告

给Lindens发送当机报告，用于修改程序中的bugs。
  • Ask before sending 发送前弹出确认框。
  • Always Send 自动发送。
  • Never send 从不发送。

Language:界面语言

建议使用默认的英文。

Input & Camera 输入和镜头页


Mouselook Options:鼠标视角

在鼠标视角状态，你看不到自己的身体，感觉象第一人称那样。向前推鼠标滚轮，或者按M键可以切换的鼠标视角状态。向后转滚轮，或者按ESC键，则切换回默认视角。

  • Mouse Sensitivity -鼠标灵敏度，
  • Invert Mouse - 翻转鼠标方向。只颠倒上下方向。

Auto Fly Options:飞行选项

按字母E、Page Up键或者是数字键盘上的9（要关掉NumLock）可以向上飞。
向下飞，用字母C、Page Down键 或 数字键盘上的3。
按字母F、Home键 或者数字键盘上的7 ，可以切换飞行状态。

  • Fly/land on holding up/down - 钩选此项，压住向上飞，则自动切换到飞行状态。压住向下飞，当接触到地面时，则取消飞行状态。不选择此项，则只能靠字母F、Home键或数字键盘上的7来切换飞行状态。

Camera Options:镜头选项

  • Camera Springiness:镜头的电影化效果
    在你快速移动身体，再突然停下来的时候。镜头会跟随并拉近，仿佛电影效果。你可以在上下飞的时候对比此值的效果。
  • Automatic Edit Camera Movement -编辑物体时，镜头自动对焦。
  • Automatic Appearance Camera Movement - 编辑身体外观时，镜头自动对焦。

Graphics tab:图像页


Run in a window:窗口显示

选择此项，程序将以窗口模式运行，否则以全屏模式运行。可以用Alt+Enter来切换两种模式。

Fullscreen Aspect Ratio:全屏下长宽比

建议钩选Auto detect自动检测，以防硬件不支持。

UI Size:界面大小

调整界面大小，包括菜单、按钮和弹出框。屏幕分辨率小的用户，应适当调小界面，以显示出整个弹出框。

Use resolution independent scale:界面大小不随分辨率改变

仅作用于全屏模式，在改变分辨率时，界面大小不受影响。

Draw Distance:渲染范围

渲染范围越大，视野越远。当然要求的带宽也更多，因为你要下载更多的物体数据。一般只在截图时调大范围。

Show Avatar in Mouselook:鼠标视角中显示自己身体

Graphics Detail tab图像细节页

Shaders:阴影

Enable Bumpmapping and Shiny -允许凹凸贴图和金属光泽。

Enable Ripple Water -水波纹效果。

  • Avatar Vertex Program -让人物渲染更自然。如果是高档显卡，可以选择此项，否则容易出现全黑的人或其他异常。

Avatar Rendering:人物渲染

  • Normal -普通效果，无任何特效。
  • Bump Mapped -凹凸贴图。衣物上会有褶皱效果。
  • Bump Mapped & Cloth -衣物褶皱效果，头发（非prim头发）和衣服随风飘动效果。（以上效果容易引起当机，如果你的硬件不够好的话，慎重选择）

Lighting Detail:光效

Sun and moon only -仅仅有阳光和月光，没有物体光效。

Nearby local lights -局部光效。发光物体会照亮四周。（加重显卡负担）

Terrain Detail:地面纹理

  • Low -低分辨
  • High -高分辨

Object Mesh Detail - 物体细腻度

向左拖动减小该值，使得圆弧物体变的粗糙有棱角，但显示速度会提高。右拖增加该值，让物体更圆润,同时也加重显卡负担。

Flexible Mesh Detail -柔性物体动作时的细腻度

Tree Mesh Detail -树木的细腻度

树木将显示出更多枝杈和树叶。

Avatar Mesh Detail -人物细腻度

主要针对身体上的附加物品，比如prim作的头发。

Adv. Graphics tab高级图像页

Anisotropic Filtering -各向异性过滤

各向异性过滤是最新型的过滤方法，主要是克服贴图中的锯齿现象。不建议选择，除非是好好显卡。

Gamma:

减小该值使得整个屏幕亮起来。
0.01最亮，2.00最黑

Nighttime Brightness:夜晚亮度

增加夜晚的亮度。4.00最亮

AGP Graphics Card:AGP显卡

如果是AGP显卡，建议钩选。

Graphics Card Memory:显存

尽可能选大值。

Fog Distance Ratio:雾化范围

较小的值，造成一种浓雾弥漫的效果。

Max. Particle Count:最大离子数

Outfit Composite Limit:组合物件限制

Audio & Video tab音频视频页

Muting:无声

  • Mute Audio - 消除所有声音和音乐。
  • Mute Audio When Window Minimized -当窗口最小化时，消除所有声音和音乐。

Streaming:流媒体

如果你有足够的带宽保证，不妨接收流媒体的音乐和影像。

  • Play Streaming Music When Available -播放音乐，如果所处区域内有的话。
  • Play Streaming Video When Available -播放电影，如果所处区域内有的话。

Sound Effects:声效控制

Wind Volume:风声控制

Footsteps Volume:脚本声

UI Volume:界面声效

比如IM来时的的ding-dings声，或者Teleport（传送）时的swooooshes 嗖声。

L$ Change Threshold:当L$的交易值大于等于此值，则发出cha-ching声提示。

Health Change Threshold:当健康值减少的速度大于等于此值，声音提示。

Doppler Effect:多普勒效应

Distance Factor:声效范围

右移则减小声效范围。

Rolloff Factor:回音

右移缩短回音时间。

Default Upload Bitrate:上传声音的品质

Chat tab聊天页

Chat Font Size:聊天字体大小

  • Small 小
  • Medium 中
  • Large 大

Chat Color:字体颜色

  • System -系统消息
  • Users -用户间对话
  • Objects -物体对话
  • Bubble -头顶显示名字的框
  • URLs -网址

Chat Console:对话

  • Fade Chat after -对话保持显示时间（秒）。
  • lines -最大行数。
  • Opacity -对话背景的透明度，1.00为不透明。

Chat Options:聊天选项

  • Chat uses full screen width -聊天显示为屏幕宽度。
  • Close chat after hitting return -用回车键转换聊天模式。
  • Arrow keys move avatar when chatting -在聊天时，可以用方向键来移动身体。否则的话，必须关闭聊天状态。
  • Show timestamps in chat -在聊天记录框里，显示每句的时间。

Bubble chat:聊天泡

聊天泡就是每人头顶显示名字的框。

  • Show chat bubbles -将对话显示在聊天泡。
  • Opacity -泡背景的透明度。

Script Errors:脚本报错

  • Show script errors and warnings as regular chat -显示脚本错误和警告。
  • Color -字体颜色。
Communication tab 通讯页

Show Online Status in Search:在search里显示在线状态

    钩选此项，别人就可以在search里看到你在线状态。

IM options:私聊选项

  • Send IM to E-mail -你不在线的时候，用Email接受别人的IM，当然你也可以在Email发IM给在线的人。
  • Include IM in Chat History -在对话记录里包含IM谈话。
  • Show timestamps in IM -在IM里显示时间戳。
  • Log Instant Messages -保存私聊记录到硬盘。每个人、组织一个文件。
  • Log Chat -保存对话记录到硬盘，文件名为"chat.txt"。
  • Show end of last IM conversation -在谈话记录里显示你最后的聊天内容，当你再此登录的时候。前提是你要钩选前面两项，在硬盘保存记录。
  • Change Path -改变保存记录的硬盘路径。

Busy Mode Response:忙状态时的自动回复内容

    当你设置了忙状态(World menu > Set Busy),框中的文字，将是你的自动回复内容。
